# Найденные баги на скриншоте

## P0 (Критические)
1. Фильтр по городу не работает - показывается объявление из Новосибирска при поиске в Москве.
2. Фильтр по категории не работает - среди объявлений с животными показывается игрушка.
3. Фильтр по цене не работает - два объявления выходят за указанный диапазон.

## P1 (Высокие)
1. Неправильная подсказка на лайк: "Не показывайте мне похожие объявления" вместо "Добавить в избранное".
2. Некорректное отображение локации при фильтрации: "Сначала из Можайска" при поиске в Москве.

## P2 (Средние)
1. В поле "Возраст" указано "Щенок" вместо "Котенок".
2. Несоответствие количества объявлений, в заголовке "19", а в фильтрах "Показать 8".
3. Большой отступ между пятым и шестым объявлениями.
4. Седьмое объявление наклонено вправо.

## P3 (Низкие)
1. Опечатка в верхней части экрана: написано "Коръера в Авито", а должно быть "Карьера в Авито".
2. Логическая ошибка в структуре фильтров: порода сразу после "Животные".
3. Лишнее уведомление об ошибке справа над оъявлениями, хотя визуально все работает.
